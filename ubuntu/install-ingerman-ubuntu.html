How to Install ingerman on Ubuntu

  • Home
  • Ubuntu
  • ingerman

Today in this tutorial, I’ll show you how you can install ingerman on Ubuntu with and without using the command line.

Introduction

Installing ingerman on Ubuntu is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall ingerman in just a few steps.

In this tutorial I’ll show you four methods to install ingerman, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install ingerman Using apt

To install ingerman in Ubuntu using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install ingerman using the apt install command. To do that use the command given below.

sudo apt -y install ingerman

The installation of ingerman might take a little bit of time depending upon your internet connection and machine configuration.

Install ingerman Using apt-get

To install ingerman using apt-get in Ubuntu, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install ingerman.

sudo apt-get -y install ingerman

Install ingerman Using Synaptic

To install ingerman using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install ingerman on your Ubuntu using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for ingerman. This will display the ingerman package in the package window, now to install ingerman, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the ingerman package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of ingerman.

If the download and installation are successful then the checkmark option will turn green.

Install ingerman Using aptitude

To install ingerman on Ubuntu using aptitude, you will need to first install Aptitude in your ubuntu. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of ingerman.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install ingerman using the command aptitude install ingerman.

sudo aptitude -y install ingerman

How to Upgrade ingerman on Ubuntu

If you want to upgrade the ingerman package on your Ubuntu to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade ingerman using Apt & Apt-get

To upgrade ingerman to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade ingerman.

sudo apt install --only-upgrade ingerman sudo apt-get install --only-upgrade ingerman

Upgrade ingerman using Aptitude

To upgrade ingerman using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade ingerman.

sudo aptitude -y full-upgrade ingerman

Upgrade ingerman using Synaptic

To upgrade ingerman using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for ingerman, now right-click ingerman, and click on the mark for upgrade option.

Now to upgrade ingerman just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall ingerman on Ubuntu

If you want to remove and uninstall ingerman from your Ubuntu then here’s how you can do that. To remove just ingerman, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove ingerman

Remove related packages, dependencies & configuration files of ingerman

To remove all the related packages, dependencies and configuration files of ingerman you can use the command given below.

sudo apt -y purge ingerman

Remove ingerman using Synaptic Package Manager

You can also uninstall ingerman using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for ingerman, right-click the ingerman package and click on the Mark for Complete Removal option. Now to uninstall ingerman click on the apply button then in summary click on apply again.

Conclusion

That is how you can install ingerman on Ubuntu using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade ingerman using the command line or using the synaptic package manager.
