How to Install libssm-dev on Ubuntu

  • Home
  • Ubuntu
  • libssm-dev

Today in this tutorial, I’ll show you how you can install libssm-dev on Ubuntu with and without using the command line.

Introduction

Installing libssm-dev on Ubuntu is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall libssm-dev in just a few steps.

In this tutorial I’ll show you four methods to install libssm-dev, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install libssm-dev Using apt

To install libssm-dev in Ubuntu using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install libssm-dev using the apt install command. To do that use the command given below.

sudo apt -y install libssm-dev

The installation of libssm-dev might take a little bit of time depending upon your internet connection and machine configuration.

Install libssm-dev Using apt-get

To install libssm-dev using apt-get in Ubuntu, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install libssm-dev.

sudo apt-get -y install libssm-dev

Install libssm-dev Using Synaptic

To install libssm-dev using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install libssm-dev on your Ubuntu using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for libssm-dev. This will display the libssm-dev package in the package window, now to install libssm-dev, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the libssm-dev package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of libssm-dev.

If the download and installation are successful then the checkmark option will turn green.

Install libssm-dev Using aptitude

To install libssm-dev on Ubuntu using aptitude, you will need to first install Aptitude in your ubuntu. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of libssm-dev.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install libssm-dev using the command aptitude install libssm-dev.

sudo aptitude -y install libssm-dev

How to Upgrade libssm-dev on Ubuntu

If you want to upgrade the libssm-dev package on your Ubuntu to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade libssm-dev using Apt & Apt-get

To upgrade libssm-dev to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade libssm-dev.

sudo apt install --only-upgrade libssm-dev sudo apt-get install --only-upgrade libssm-dev

Upgrade libssm-dev using Aptitude

To upgrade libssm-dev using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade libssm-dev.

sudo aptitude -y full-upgrade libssm-dev

Upgrade libssm-dev using Synaptic

To upgrade libssm-dev using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for libssm-dev, now right-click libssm-dev, and click on the mark for upgrade option.

Now to upgrade libssm-dev just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall libssm-dev on Ubuntu

If you want to remove and uninstall libssm-dev from your Ubuntu then here’s how you can do that. To remove just libssm-dev, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove libssm-dev

Remove related packages, dependencies & configuration files of libssm-dev

To remove all the related packages, dependencies and configuration files of libssm-dev you can use the command given below.

sudo apt -y purge libssm-dev

Remove libssm-dev using Synaptic Package Manager

You can also uninstall libssm-dev using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for libssm-dev, right-click the libssm-dev package and click on the Mark for Complete Removal option. Now to uninstall libssm-dev click on the apply button then in summary click on apply again.

Conclusion

That is how you can install libssm-dev on Ubuntu using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade libssm-dev using the command line or using the synaptic package manager.
