How to Install python-configargparse on Ubuntu

  • Home
  • Ubuntu
  • python-configargparse

Today in this tutorial, I’ll show you how you can install python-configargparse on Ubuntu with and without using the command line.

Introduction

Installing python-configargparse on Ubuntu is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall python-configargparse in just a few steps.

In this tutorial I’ll show you four methods to install python-configargparse, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install python-configargparse Using apt

To install python-configargparse in Ubuntu using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install python-configargparse using the apt install command. To do that use the command given below.

sudo apt -y install python-configargparse

The installation of python-configargparse might take a little bit of time depending upon your internet connection and machine configuration.

Install python-configargparse Using apt-get

To install python-configargparse using apt-get in Ubuntu, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install python-configargparse.

sudo apt-get -y install python-configargparse

Install python-configargparse Using Synaptic

To install python-configargparse using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install python-configargparse on your Ubuntu using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for python-configargparse. This will display the python-configargparse package in the package window, now to install python-configargparse, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the python-configargparse package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of python-configargparse.

If the download and installation are successful then the checkmark option will turn green.

Install python-configargparse Using aptitude

To install python-configargparse on Ubuntu using aptitude, you will need to first install Aptitude in your ubuntu. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of python-configargparse.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install python-configargparse using the command aptitude install python-configargparse.

sudo aptitude -y install python-configargparse

How to Upgrade python-configargparse on Ubuntu

If you want to upgrade the python-configargparse package on your Ubuntu to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade python-configargparse using Apt & Apt-get

To upgrade python-configargparse to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade python-configargparse.

sudo apt install --only-upgrade python-configargparse sudo apt-get install --only-upgrade python-configargparse

Upgrade python-configargparse using Aptitude

To upgrade python-configargparse using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade python-configargparse.

sudo aptitude -y full-upgrade python-configargparse

Upgrade python-configargparse using Synaptic

To upgrade python-configargparse using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for python-configargparse, now right-click python-configargparse, and click on the mark for upgrade option.

Now to upgrade python-configargparse just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall python-configargparse on Ubuntu

If you want to remove and uninstall python-configargparse from your Ubuntu then here’s how you can do that. To remove just python-configargparse, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove python-configargparse

Remove related packages, dependencies & configuration files of python-configargparse

To remove all the related packages, dependencies and configuration files of python-configargparse you can use the command given below.

sudo apt -y purge python-configargparse

Remove python-configargparse using Synaptic Package Manager

You can also uninstall python-configargparse using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for python-configargparse, right-click the python-configargparse package and click on the Mark for Complete Removal option. Now to uninstall python-configargparse click on the apply button then in summary click on apply again.

Conclusion

That is how you can install python-configargparse on Ubuntu using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade python-configargparse using the command line or using the synaptic package manager.
