How to Install saga on Ubuntu

  • Home
  • Ubuntu
  • saga

Today in this tutorial, I’ll show you how you can install saga on Ubuntu with and without using the command line.

Introduction

Installing saga on Ubuntu is very easy. And with the help of this tutorial, you will be able to easily install, update and uninstall saga in just a few steps.

In this tutorial I’ll show you four methods to install saga, the methods are using apt, apt-get, aptitude, and synaptic. All these methods require you to have an active and stable internet connection and sudo permission.

Install saga Using apt

To install saga in Ubuntu using apt, you will need to run the apt update command first to get the updated list of packages from the repositories.

Copy the command given below and paste it in the command line, then press enter. If you are using the sudo command for the first time then you might also need to enter your root password.

sudo apt update

After you have updated the package list using apt, you can now install saga using the apt install command. To do that use the command given below.

sudo apt -y install saga

The installation of saga might take a little bit of time depending upon your internet connection and machine configuration.

Install saga Using apt-get

To install saga using apt-get in Ubuntu, you will need to first update the package list using the apt-get update command. Copy, paste, and enter the command given below in the command line.

sudo apt-get update

After updating the package lists, use the apt-get install command given below to install saga.

sudo apt-get -y install saga

Install saga Using Synaptic

To install saga using Synaptic you will need to first install Synaptic, you can do that by following the instructions from here. After installing synaptic follow the steps below to install saga on your Ubuntu using Synaptic.

You can open synaptic by clicking on the menu icon then searching for synaptic or you can open the command line and enter the command given below –

sudo synaptic

After synaptic opens, click on the search icon and search for saga. This will display the saga package in the package window, now to install saga, click on the checkbox that is on the left side.

Then click on Mark for Installation, this will select the saga package for installation, now to install it click on the Apply button, you will get a summary window where click on the apply button again to start the download of saga.

If the download and installation are successful then the checkmark option will turn green.

Install saga Using aptitude

To install saga on Ubuntu using aptitude, you will need to first install Aptitude in your ubuntu. To install Aptitude follow the instructions here. Once you have aptitude installed proceed to the installation of saga.

Now you will need to first update your package lists from the repositories, use the apt update command for that, after that update the package list of aptitude using the aptitude update command.

sudo apt update sudo aptitude update

After updating the package list of aptitude, install saga using the command aptitude install saga.

sudo aptitude -y install saga

How to Upgrade saga on Ubuntu

If you want to upgrade the saga package on your Ubuntu to its latest version then here’s how you can do that using apt, apt-get, aptitude, and synaptic.

Upgrade saga using Apt & Apt-get

To upgrade saga to its latest version using apt or apt-get, you will need to open the terminal then run the sudo apt update command to update the package list, after that copy and paste the commands given below to upgrade saga.

sudo apt install --only-upgrade saga sudo apt-get install --only-upgrade saga

Upgrade saga using Aptitude

To upgrade saga using aptitude you will need to again run the update command to update the package list, use command sudo aptitude update to do that. After that run the command given below to upgrade saga.

sudo aptitude -y full-upgrade saga

Upgrade saga using Synaptic

To upgrade saga using synaptic open synaptic using the command sudo synaptic or by going to the menu and searching for synaptic.

In synaptic click on the reload button, this will look for updates. After that you will need to search for saga, now right-click saga, and click on the mark for upgrade option.

Now to upgrade saga just click on the apply button, then click on the apply button again in the summary screen.

How To Uninstall saga on Ubuntu

If you want to remove and uninstall saga from your Ubuntu then here’s how you can do that. To remove just saga, you can use the command that is given below, copy the command, paste it in the command line then press enter.

sudo apt remove saga

Remove related packages, dependencies & configuration files of saga

To remove all the related packages, dependencies and configuration files of saga you can use the command given below.

sudo apt -y purge saga

Remove saga using Synaptic Package Manager

You can also uninstall saga using the synaptic package manager, first open synaptic using the command sudo synaptic or search and open it directly.

After that in synaptic search for saga, right-click the saga package and click on the Mark for Complete Removal option. Now to uninstall saga click on the apply button then in summary click on apply again.

Conclusion

That is how you can install saga on Ubuntu using the command line or using the synaptic package manager. I also showed you how you can uninstall and upgrade saga using the command line or using the synaptic package manager.
